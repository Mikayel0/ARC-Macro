# Macro (Mikayel Macro v1.0)

A powerful, customizable macro recording and playback tool designed for precision control and automation.

## Requirements

- **.NET 10 Runtime**: This application is built using .NET 10. You will need to install the .NET 10 Desktop Runtime to run it. 
  - [Download .NET 10 here](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)

## Workflow (How to Use)

Using the Macro tool involves three simple steps:

1. **Record a Macro**
   - Click "Toggle Record" or press `[F8]` (or your custom bind) to start recording your keyboard/mouse actions.
   - Perform your actions.
   - Press the key again to stop recording.

2. **Save the Macro**
   - After recording, the macro resides in memory.
   - Enter a name in the bottom text box and click "Save Recorded Macro". 
   - It will now appear in your "Saved Macros" list.

3. **Play the Macro**
   - Select your saved macro from the list.
   - Click "Play/Stop Macro" or press `[F9]` (or your custom bind) to execute it.
   - You can pause or stop playback at any time using the corresponding hotkey.

## Additional Features
- **Adjustable Speed**: Change playback speed using the slider on the main UI.
- **Overlay Engine**: Activate an on-screen overlay border while working.
- **Audio Mute**: Auto-mute specific game/app audio during macro execution if configured.
